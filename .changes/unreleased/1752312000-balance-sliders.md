type: Changed
title: Developer balance sliders
---

The developer BALANCE page now drives each knob with an exponential slider (drag, tap, or ←/→) spanning 0× (system off) to 100× the shipped tuning — where 1× is baseline — replacing the preset 25%–400% percentage steps; the track's four quarters cover 0→1, 1→2, 2→10, and 10→100 so the useful low end gets most of the travel.
