---
type: Added
title: Import and export characters
---

**SETTINGS → DATA → EXPORT / IMPORT CHARACTER** carry heroes between devices: EXPORT opens a picker over the whole roster where you tick one or many heroes and download each as a small signed `.zip` (an HMAC-SHA256 signature over the save makes casual tampering fail), and IMPORT loads one back through a file picker — bringing its level, gear, and campaign progress across.
