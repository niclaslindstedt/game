title: Static sheets show the stride, not the motion — and lie about context
date: 2026-07-11

`variants`/`concepts` prove `_0`/`_1` both read and that wounds survive, but
ground separation, hierarchy next to the hero, and the walk cadence only
show in the running game. The scenario pose check (Phase 4 step 7) is the
cheap way to close that loop for anything stageable — a frozen screenshot
for the read, an unfrozen run for the motion; don't trust the still frames
alone, and don't hand-play the game to reach the sprite either.
